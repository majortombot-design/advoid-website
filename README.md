# Advoid Website

## Files
- `index.html` - Main page (EDIT THIS)
- `style.css` - Styles (EDIT COLORS/FONTS HERE)

## To Edit Content
1. Open `index.html` in any text editor
2. Find the section you want to edit (look for comments like `<!-- Hero Section -->`)
3. Change the text between tags
4. Save and refresh browser

## To Deploy (Free)
1. Push to GitHub
2. Enable GitHub Pages in repo settings
3. Or use Netlify drag-and-drop

## Sections to Customize
- Hero headline (line 25)
- Services descriptions (lines 45-60)
- Work portfolio items (lines 70-90)
- About text (lines 100-110)
- Contact email (line 125)
